🏥 SmartHealth — Health Prediction & Appointment System

📌 Project Overview

SmartHealth is a full-stack web-based healthcare application designed to help users predict diseases based on symptoms and connect with doctors for consultation.

The system provides an intelligent health assistance platform with prediction, doctor management, and appointment booking features.


🚀 Features

- User Registration & Login
- Doctor Login
- Symptom-based Disease Prediction
- Book Appointment with Doctors
- Multiple Doctor Management
- User Dashboard
- Doctor Dashboard
- Health Records Management

🏗️ System Architecture

Frontend  →  Backend  →  Database  →  ML Module
 (UI)        (Logic)      (MySQL)      (Integrated separately)

⚙️ Technologies Used

💻 Frontend

- HTML
- CSS
- JavaScript
- JSP / Servlets

🧠 Backend

- Java (NetBeans)

🗄️ Database

🔮 Machine Learning

🧪 Modules

1️⃣ User Module

- Registration
- Login
- Symptom entry

2️⃣ Prediction Module

- Sends symptoms for analysis
- Displays predicted disease

3️⃣ Doctor Module

- Doctor login
- Manage appointments
  
4️⃣ Appointment Module
- Book / View appointment
